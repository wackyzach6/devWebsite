Task 1.6 Describing access issues that need taking into account - describe issues that people with disabilities may face when using webpages and how we can overcome them

Some issues that may arise for easy accessibility are that some people using the site may be unable to read or see well so the website will need any images that contain text to have an alternative text as part of the image so that when people have a software that can read the webpage it will detect the text in the image so users will know what is on the website.
Another issue is that any audio would need subtitles in the video or within the webpage for any users with problems hearing so that they know what is being said on the website.
Another Issue is that some people will need a larger font to read the text on the webpage

Task 1.7 Describe which are the appropriate filetypes for websites

The common appropriate file type for webpages are HTML or the webpage, CSS for some of the organising of the webpage, PNG, JPG and JPEG for some images, GIF for any GIF’s, MP4 for videos and MP3 for audio files as these are the most commonly used file types.
